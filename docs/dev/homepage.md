### Заметка из дизайна

> Графика и живопись — по годам. Просто папки, внутри папок файлы. Без описаний, техник и прочего. Написать js-скрипт, который всё это вытаскивает наверх и формирует страницу из девайдеров.
> Code — по проектам. К каждому проекту — описание с ссылками и презентацией. БОЛЬШИЕ КАРТОЧКИ. Резюме + CV. С возможностью поделиться.
> DigitalArt — аналогично живописи.
> Фотография — аналогично живописи.
> About — лента из социалок + вставки из биографии + резюме + CV.

### Заметки из дизайна

> `_directory-name` — создаётся девайдер. Отрисовываются внутренние блоки на внешней странице. Вешается класс по названию директории. В дочерней директории отрисовывается в `hidden`.

> Для WIP-проектов обозначить их серым цветом. Disabled.

> Для xtc-toaster вытащить из легаси генератор постеров. Опубликовать пакет/docker. Перебрать документацию.

> Для каждой карточки давать уникальный айдишник, чтобы можно было собрать в отдельную сетку.
